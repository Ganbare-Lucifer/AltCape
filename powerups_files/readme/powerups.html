Hammer Suit (Super Mario Bros. 3)


Description
The iconic Hammer Suit from Super Mario Bros. 3 is available in this powerup pack. With it Mario can throw hammers that describes an arc movement. Their horizontal speed is adjusted by how fast is Mario going.The hammers also are super powerful, you can defeat enemies with a single hammer, unless they are immune to them!


Controls
  • X/Y: Shoot hammer.
  • A: Shoot hammers while spin-jumping.

Customization
The powerup is highly customizable, you can go to /powerups_files/powerup_defs folder and open powerup_04.asm. Please take a good read of it before changing anything there!
You can also change the projectile graphics in the extended sprites folder.
There are a few blocks that works with this powerup inside of the GPS folder.

Super Leaf (Super Mario Bros. 3)


Description

Another iconic powerup is included here! The Super Leaf from Super Mario Bros. 3 is fully functional. The powerup lets Mario to fly for a short period of time, it also gives him a more accurate Tail attack and animation.



Controls

  • X/Y: Spin attack.
  • X/Y in air: Spin attack.
  • Run + B: Start flying.
  • While flying, tap B: Go higher.
  • While descending, tap B: Float in air.

Customization

The powerup is highly customizable, you can go to /powerups_files/powerup_defs folder and open powerup_06.asm. Please take a good read of it before changing anything there!
There are a few blocks that works with this powerup inside of the GPS folder.

Tanooki Suit (Super Mario Bros. 3)


Description

The Tanooki Suit powerup makes an appearance here, it lets Mario do the same stuff as if he had a Super Leaf, but with the exception that he also can convert himself into a statue to become invincible and crush enemies!



Controls
  • X/Y: Spin attack.
  • X/Y in air: Spin attack.
  • Run + B: Start flying.
  • While flying, B: Go higher.
  • While descending, B: Float in air.
  • D-Pad down + tap X/Y: Transform into statue.
  • In air, D-Pad down + tap X/Y: Statue drop.
  • While being in statue form, D-Pad down: Keep statue form.
  • While being in statue form, release all buttons: Revert statue form.

Customization

The powerup is highly customizable, you can go to /powerups_files/powerup_defs folder and open powerup_07.asm. Please take a good read of it before changing anything there!
There are a few blocks that works with this powerup inside of the GPS folder.
