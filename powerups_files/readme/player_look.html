Changing Player's palette


Changing the player palette can't be done via Lunar Magic, but it's quite easy to do, just be sure to follow the next steps:

1) Go to powerups_files/powerup_misc_data/ and open palette.asm



2) Once you open it, you will be welcomed with a lot of tables with strange numbers. These 16-bit values corresponds to the SNES RGB value.
 NOTE: The patch only modifies colors 6 to F!

3) Open Lunar Magic's palette editor and build your palette there, when it is completed, check the field at the bottom of the window:



4) Copy the SNES RGB values that Lunar Magic gives to you in the desired table.

5) Apply the main patch again and check the changes in-game!

Changing Player's graphics


The player graphics are easily changeable, it doesn't matter if you use this patch or not.

1) Go to powerups_files/graphics/



2) Open any of the .bin files that are given to you in that folder and modify them as you wish.

If you haven't noticed, this is almost the same as modifying the player graphics in the original game.
